# Kubernetes EKS Nginx Deployment

## Project Overview
This project demonstrates deploying an Nginx application on AWS EKS using Kubernetes.

## Steps Performed
1. Created EKS cluster using eksctl
2. Configured kubectl to connect cluster
3. Deployed Nginx using deployment YAML
4. Exposed application using LoadBalancer service
5. Accessed application via external URL

## Output
Nginx successfully running on AWS EKS and accessible via browser.
